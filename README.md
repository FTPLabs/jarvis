## Что такое JARVIS?

JARVIS — настольный голосовой ИИ-ассистент, работающий **полностью локально**.
Никаких облаков, никакой слежки. Вдохновлён персонажем из «Железного человека», адаптирован для русского языка.

## 🔒 Конфиденциальность

**100% анонимность** — все данные обрабатываются на вашем компьютере.
- Голос не записывается и не отправляется
- AI работает через локальный Ollama (llama3:8b)
- Никакой телеметрии и аналитики

Полная политика: [PRIVACY.md](PRIVACY.md)

## Возможности

- Голосовое управление — скажите «Джарвис» и отдайте команду
- Локальный AI — Ollama + llama3:8b без интернета
- Самообучение — запоминает ваши предпочтения
- Запуск приложений — Chrome, Telegram, Spotify и другие
- Поиск — голосовой поиск в Google и YouTube
- Статистика — активность, YouTube/TikTok аналитика
- Системный трей — работает в фоне

## Быстрый старт (Windows)

1. Скачайте `JARVIS-Setup-x.x.x.exe` с [GitHub Releases](https://github.com/FTPLabs/jarvis/releases)
2. Установите (SmartScreen может показать предупреждение — нажмите «Всё равно запустить»)
3. Установите [Ollama](https://ollama.ai/) и выполните: `ollama pull llama3:8b`
4. Скажите **«Джарвис»** или нажмите **Ctrl+Shift+J**

**Демо-ключ для тестирования:** `JARVIS-DEMO-TEST-2024`

## Системные требования

- **ОС:** Windows 10/11 (64-bit), macOS 12+, Linux
- **ОЗУ:** минимум 8 ГБ (рекомендуется 16 ГБ)
- **Диск:** 10 ГБ (для AI моделей)
- **Интернет:** не требуется
