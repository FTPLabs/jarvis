  Frontend (UI):
    React + Vite + TypeScript 5.9
    Tailwind CSS + Framer Motion (анимации)
    Recharts (графики статистики)
    Папка: artifacts/jarvis-ui/

  Backend API:
    Express 5 + Node.js 24
    Drizzle ORM + PostgreSQL
    Папка: artifacts/api-server/

  Desktop:
    Electron (обёртка для Windows/macOS/Linux)
    Папка: electron/

  AI / Голос (Python):
    FastAPI — Python-ядро
    Whisper — распознавание речи (STT), русский язык
    Piper TTS — синтез речи на русском
    Ollama + llama3:8b — локальный LLM (без облака)
    Porcupine (опционально) — обнаружение wake word "Джарвис"
    Папка: desktop/core/

  Формат пробной лицензии    : JARVIS-XXXX-XXXX-XXXX  (30 дней)
  Формат постоянной лицензии : JARVIS-PERM-XXXX-XXXX  (навсегда)

  Файл лицензии хранится: ~/.jarvis/license.json

  ПЕРВОНАЧАЛЬНАЯ УСТАНОВКА (WINDOWS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. Скачайте JARVIS-Setup-1.0.0.exe с GitHub Releases
  2. Запустите установщик (может потребоваться "Всё равно запустить" в SmartScreen)
  3. JARVIS установится в %LOCALAPPDATA%\JARVIS\
  4. Добавится в автозапуск Windows автоматически
  5. Откроется в системном трее (иконка в правом нижнем углу)
  6. Установите Ollama: https://ollama.ai/
  7. В командной строке: ollama pull llama3:8b
  8. Скажите "Джарвис" или нажмите Ctrl+Shift+J

  Требования к системе:
    ОС: Windows 10 / 11 (64-bit)
    ОЗУ: минимум 8 ГБ (рекомендуется 16 ГБ)
    Диск: 10 ГБ свободного места (для AI моделей)
    Интернет: не требуется (всё работает локально)

  САМООБУЧЕНИЕ — КАК РАБОТАЕТ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  JARVIS автоматически учится из:
  • Каждого разговора — хранит историю и анализирует темы
  • Имя и факты о пользователе — извлекает из фраз
  • Предпочтения — запоминает что вы любите/не любите
  • Часто используемые приложения — смотрит на что вы переключаетесь
  • Время активности — знает в какие часы вы работаете
  • Цели — запоминает ваши планы ("хочу", "планирую", "собираюсь")

  Данные хранятся локально:
    ~/.jarvis/memory.json    — факты, имя, предпочтения, цели
    ~/.jarvis/learning.json  — история разговоров, частота команд
    ~/.jarvis/activity.json  — активность экрана, приложения
    ~/.jarvis/jarvis.log     — системный лог
    ~/.jarvis/license.json   — лицензия (HWID-привязанная)

  Никакие данные НЕ отправляются в интернет — всё остаётся на вашем ПК.
